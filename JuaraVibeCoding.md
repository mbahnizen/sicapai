# #JuaraVibeCoding — Ringkasan Event

## Tentang Event

**#JuaraVibeCoding** adalah program dari Google for Developers yang mengajak siapa saja membangun aplikasi menggunakan AI tanpa harus punya pengalaman coding.

Peserta akan belajar:

* 🧠 Brainstorm dengan Gemini
* 🛠️ Build menggunakan Google AI Studio
* ☁️ Deploy aplikasi ke Cloud Run

> “If you can dream it, you can vibe-code it.”

* 📍 Format: Online (Indonesia)
* 📅 Start Date: 6 Mei 2026
* 📅 Deadline Submission: 31 Mei 2026

---

# Kenapa Ikut #JuaraVibeCoding?

* 🚀 Free Google Cloud Credits
* 🤝 Bergabung dengan ribuan developer Indonesia di Discord
* 💡 Belajar prototyping AI menggunakan AI Studio & Cloud Run
